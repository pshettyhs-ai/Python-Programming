## 📖 About This Repository

I'm an Embedded Systems enthusiast who realized that Python is something I absolutely
cannot ignore. Whether it's automating repetitive tasks, working with APIs for IoT
projects, or just writing cleaner logic — Python kept showing up everywhere.

So I decided to go deep. This repository is my structured attempt to master Python
from scratch to a level where I can confidently use it in real projects and interviews.

I started this in January 2024 and have been adding to it consistently. Some modules
took me a weekend. The OOP section took me almost three weeks because I kept confusing
myself with inheritance chains. The Database section was where things finally started
feeling "real" to me.

**What makes this different from a tutorial repo:**
- Notes written the way I actually understood things, not how textbooks explain them
- Mistakes I made are documented (sometimes in comments, sometimes in the Notes.md)
- Projects evolved — messy first versions exist, cleaned-up ones are the final commits
- Interview questions are ones I actually got asked or found while preparing

## ⚙️ Installation & Setup

```bash
# 1. Clone the repository
git clone https://github.com/PavanShettyHS/Python-Programming-Mastery.git

# 2. Move into the directory
cd Python-Programming-Mastery

# 3. Create a virtual environment (recommended)
python -m venv venv

# 4. Activate it
# On Linux/Mac:
source venv/bin/activate
# On Windows:
venv\Scripts\activate

# 5. Install dependencies
pip install -r requirements.txt

# 6. Run any file
python 01_Python_Basics/HelloWorld.py
```
